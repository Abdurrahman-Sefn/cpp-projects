# Ask Me System

This is a console-based question and answer system.

## Description

This program allows users to ask and answer questions. It's a simplified, text-based version of a platform like ask.fm. Users can register, log in, ask questions, answer questions, and view questions and answers.

## Building and Running

To build the project, you can use the following command:

```bash
g++ main.cpp AskMe.cpp QuestionsManager.cpp UsersManager.cpp Question.cpp User.cpp Helper.cpp -o program.out
```

To run the project, execute the following command:

```bash
./program.out
```

## Usage

Upon running the program, you will be presented with a menu to either log in or sign up. Once logged in, you can navigate through various options to manage questions and answers.
