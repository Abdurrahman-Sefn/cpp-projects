# Tic Tac Toe Game

This is a simple console-based Tic Tac Toe game.

## Description

This program allows two players to play a game of Tic Tac Toe. The game board is displayed in the console, and players take turns entering their moves. The game ends when one player wins or the board is full.

## Building and Running

To build the project, you can use the following command:

```bash
g++ program.cpp -o program
```

To run the project, execute the following command:

```bash
./program
```

## Features

Upon running the program, the game will start automatically. Players take turns entering their moves by specifying the row and column of the cell they want to mark.
