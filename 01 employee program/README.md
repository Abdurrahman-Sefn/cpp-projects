# Employee Program

This is a simple console-based employee management system.

## Description

This program allows you to add, delete, and view employee data. The employee data is stored in a file named `data.txt`.

## Building and Running

To build the project, you can use the following command:

```bash
g++ system.cpp -o system
```

To run the project, execute the following command:

```bash
./system
```

## Features

Upon running the program, you will be presented with a menu of options:

1.  Add new employee
2.  Print all employees
3.  Delete by age
4.  Update salary by name
5.  End program
