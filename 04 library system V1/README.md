# Library System V1

This is a console-based library management system.

## Description

This program allows you to manage books and users in a library. You can add new books and users, borrow and return books, and search for books.

## Building and Running

To build the project, you can use the following command:

```bash
g++ main.cpp LibrarySystem.cpp BooksManager.cpp UsersManager.cpp Book.cpp User.cpp Helper.cpp -o program.out
```

To run the project, execute the following command:

```bash
./program.out
```

## Usage

Upon running the program, you will be presented with a menu of options to manage the library.

## Version Information
