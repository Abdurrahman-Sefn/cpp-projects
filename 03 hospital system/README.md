# Hospital System

This is a simple console-based hospital management system.

## Description

This program allows you to manage patients in a hospital. You can add new patients, print the list of patients, and manage patient specializations. The patient data is stored in a file named `patients.txt`.

## Building and Running

To build the project, you can use the following command:

```bash
g++ hospital_system.cpp -o hospital_system
```

To run the project, execute the following command:

```bash
./hospital_system
```

## Usage

Upon running the program, you will be presented with a menu of options:

1.  Add new patient
2.  Print all patients
3.  Get next patient
4.  Exit
